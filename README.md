# product-catalog
Мини-приложение на React для просмотра каталога товаров и добавления их в корзину. Адаптивная вёрстка (desktop, tablet, mobile), Dark mode.

## Использование

### Установка

```
npm install
```

### Настройка переменных среды
Требуется указать переменную среды VITE_BE_URL, которая содержит ссылку на API приложения

### Запуск

```
npm run build
npm run preview
```

или

```
npm run dev
```

## Возможности
В каталоге реализованы:
- Фильтрация и поиск: поиск по названию товара (с debounce), фильтр по категории, сортировка (по цене, по названию);
- Пагинация с показом по 12 товаров на странице;
- Disabled состояние для кнопок;
- Skeleton при загрузке данных;
- Toast-уведомления при добавлении в корзину.

Функциональность корзины:
- Отображение добавленных товаров;
- Изменение количества товара (+ / -); 
- Удаление товара из корзины;
- Подсчёт общей стоимости;
- Иконка корзины с badge (количество товаров);
- Сохранение корзины в localStorage.
